Dust Busters Inc.

Made in Unity

Dust Busters Inc. is a haunted cleaning simulator game I am currently making as part of a team. The player is tasked with cleaning a house while the ghost that haunts it tries to prevent the player from cleaning.

My Role: Lead Programmer

My responsibilities:

  • Review scripts to ensure they are held to a professional standard.
  • Investigate and resolve bugs that are found in testing.
  • Program the ghost's AI and behaviour according to the game designer's specifications
  • Create and maintain the debug tool.
  • Maintain the GitHub repo.

    • The Ghost

    The ghost is the game's primary antagonist and the main obstacle the player faces. Multiple systems needed to be implemented for the ghost to perform its role properly.

  • I implemented an aggression level system, with different interaction target lists for each aggression level.
  • I used Unity's NavMesh tool and toggleable NavMeshObstacles to make the ghost avoid being spotted at early aggression levels.
  • I made the ghost's task list heavily customizable in the unity inspector for ease of customization.

    • The Debug Tool

    The debug tool was created to make it easier to test the game. While active, the debug screen gives the tester more control over the game to make it easier to test specific aspects of the game.

  • Can set the ghost's aggression level.
  • Can freeze the ghost for when they need to test a player task without ghost interference.
  • Can see the ghost's task list and their current task.
  • Provides a minimap that shows the ghost's and the player's positions.
